Frame Format
Use \r\n for CR+LF, \n for LF, \r for CR.
Parsing
The first capture group must match the numeric weight value.
If the line matches this pattern the weight is marked stable. Leave blank to always mark stable.
1 = scale sends kg  ·  0.001 = scale sends grams  ·  1000 = scale sends tonnes
